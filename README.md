# Pet-проект: автотесты для Яндекс.Маркета

## Описание проекта
Проект содержит UI-автотесты на Java + Selenium WebDriver для проверки фильтрации и поиска товаров на сайте Яндекс.Маркет.

Тестовый сценарий реализован в соответствии с заданием BellIntegrator.

---

## Стек
- Java 17
- Selenium WebDriver
- JUnit 5
- Maven
- Allure

---

## Что проверяет тест
Автотест выполняет следующий сценарий:

1. Открывает браузер и переходит на сайт Яндекс.Маркет
2. Открывает каталог
3. Переходит в раздел **Электроника**
4. Открывает раздел **Ноутбуки**
5. Проверяет, что переход в раздел выполнен успешно
6. Устанавливает фильтр по цене: **от 10000 до 20000 рублей**
7. Устанавливает фильтр по производителям: **HP** и **Lenovo**
8. Проверяет, что на первой странице отображается более **7** товаров
9. Проверяет, что все найденные товары соответствуют выбранным фильтрам
10. Возвращается в начало списка и запоминает название первого товара
11. Вводит это название в строку поиска
12. Выполняет поиск
13. Проверяет, что искомый товар присутствует в результатах поиска

---

## Структура проекта
Проект организован по паттерну **Page Object Model**.

### Основные части:
- `tests` — тестовые классы
- `pages` — page object классы с локаторами
- `steps` — шаги взаимодействия со страницами
- `data` — тестовые данные

---

## Особенности реализации
- Использована параметризация тестов через `@ParameterizedTest`
- Для динамической подгрузки товаров реализован плавный скролл страницы
- Для фильтров реализована проверка соответствия товаров по:
    - цене
    - производителю
- Для поиска реализована проверка наличия выбранного товара в выдаче
